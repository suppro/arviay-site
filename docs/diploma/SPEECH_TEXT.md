## Вступление (30 секунд)

Добрый день, уважаемые члены государственной аттестационной комиссии!

Вашему вниманию представляется выпускная квалификационная работа на тему: **"Разработка и внедрение интернет-магазина для АО «Арвиай»"**.

Актуальность работы обусловлена необходимостью автоматизации процессов продажи авиационных запчастей. Традиционные методы работы с клиентами создавали высокую нагрузку на менеджеров и ограничивали возможности масштабирования бизнеса.

**Цель работы** – разработка и внедрение интернет-магазина для автоматизации процессов приема и обработки заказов с интеграцией в систему учета 1С:Предприятие.

---

## Технологический стек и архитектура (1 минута)

Для реализации проекта выбран следующий технологический стек:

- **Backend:** Laravel 12 и PHP 8.2+ – обеспечивает полный контроль над архитектурой и нативную поддержку JSON
- **База данных:** MySQL 8.0 – с поддержкой JSON для технических характеристик товаров
- **Frontend:** Blade, Tailwind CSS, JavaScript – для создания адаптивного интерфейса
- **Интеграция:** REST API для взаимодействия с системой 1С:Предприятие

Архитектура системы построена на паттерне MVC и включает пять основных модулей: каталог товаров, корзина и оформление заказа, управление заказами, административная панель и модуль интеграции с 1С.

База данных спроектирована с учетом нормализации до третьей нормальной формы и включает шесть основных таблиц.

---

## Реализация системы (1,5 минуты)

**Клиентская часть** обеспечивает:

- Каталог товаров с иерархической структурой категорий, интеллектуальным поиском и системой фильтрации
- Корзину покупок с управлением количеством товаров
- Трехшаговый процесс оформления заказа с валидацией данных на каждом шаге
- Личный кабинет с историей заказов и отслеживанием статусов

**Административная панель** предоставляет инструменты для:

- Управления товарами, категориями, заказами и пользователями
- Просмотра аналитики и статистики
- Управления интеграцией с 1С:Предприятие

**Интеграция с 1С:Предприятие** реализована через REST API и обеспечивает:

- Автоматическую выгрузку заказов при изменении статуса
- Синхронизацию данных в реальном времени
- Обработку ошибок и механизм повторной выгрузки

Это исключает необходимость ручного ввода данных и снижает количество ошибок.
